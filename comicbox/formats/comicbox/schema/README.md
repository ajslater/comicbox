# Comicbox Fields

Most comicbox fields capture concepts from more than one format. These fields
are unique to one format.

## Fields unique to one format

### Comicbox

- critical_rating (no longer mapped to any format; ratings now flow through
  community_rating)

### CoMet

- cover_image
- last_mark
- reading_direction
- rights

### ComicBookInfo

- credit_primaries

### ComicInfo

- manga
- monocrhrome
- pages
- protagonist
- review
- series_groups

### MetronInfo

- alternative_issue
- community_rating.rating_count
- identifier_primary_source
- store_date
- universes

### Filename

- ext
- remainders
