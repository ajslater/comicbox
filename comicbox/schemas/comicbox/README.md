# Comicbox Fields

Most comicbox fields capture concepts from more than one format. These fields
are unique to one format.

## Fields unique to one format

### CoMet

- cover_image
- last_mark
- reading_direction
- rights

### ComicBookInfo

- credit_primaries
- critical_rating

### ComicInfo

- manga
- monocrhrome
- pages
- protagonist
- review
- series_groups

### MetronInfo

- identifier_primary_source
- store_date
- universes

### ComicTagger

- alternate_images
- series_id
- issue_id

#### ComicTagger and one other format

- cover_image (CoMet)
- last_mark (CoMet)
- monochrome (ComicInfo)
- pages (ComicInfo)
- rights (CoMet)
- series_groups (ComicInfo)
